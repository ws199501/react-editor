# react-editor
编辑器
* 图片选择后的储存和显示需要自己配置。
* 组件核心 html5标签 和 document.execCommand()，文档地址：https://developer.mozilla.org/zh-CN/docs/Web/API/Document/execCommand
* 有非常多不足的地方。内容可以自己参考去丰富，优点是使用方法十分简单。
